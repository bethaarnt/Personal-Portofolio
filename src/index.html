Tentang Saya

Saya Betha Rianti, Saya adalah Mahasiswa Teknik Informatika Universitas Sriwijaya angkatatan 2020. Saya Memiliki beberapa latar balakang di bidang IT, yaitu Pengembangan Aplikasi Web (Web Developer), UI/UX Design, dan IT Support seperti Instalasi, Perawatan dan Perbaikan Sistem Operasi.

Saya pernah berkontribusi sebagai Front-end Web Developer di salah satu komunitas kecil, yaitu Google Developer Students Club Universitas Sriwijaya. Dan saat ini saya sedang mempelajari lebih dalam tentang Pengembangan aplikasi Web, khususnya JavaScript.

Disela-sela kesibukan, saya belajar mengenai Pengembangan Aplikasi Web. saya juga sering menyempatkan diri untuk Belajar Bahasa Inggris. Selain karena Bahasa Inggris Adalah Bahasa Internasional, Bahasa Inggris Juga sangat berperan dalam Jenjang karir saya kedepannya.
